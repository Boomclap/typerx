# typerx

一个轻量带使用注解方式API方式的全栈系统开发系统。

[English README](README.md)

# 项目缘由




### Quick start

```bash
# clone our repo
# --depth 1 removes all but one .git commit history
$ git clone --depth 1 https://github.com/vellengs/typerx.git

# change directory to our repo
$ cd typerx

# install the repo with npm
$ npm install

# start the server
$ npm start

```

### Other commands

```bash

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```

### Test

```
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
